# كيك العرب — موقع تعريفي

موقع تسويقي عربي (RTL) لمحل **كيك العرب** لكيك وحلويات المناسبات.

## المعلومات
- **الاسم:** كيك العرب
- **التصنيف:** محل كيك وحلويات مناسبات
- **المدينة:** خميس مشيط — حي النسيم
- **التقييم:** 4.5 على خرائط قوقل (66 مراجعة)
- **الجوال:** 0509300466 · **واتساب:** 966509300466

## سبب الاختيار
نشاط بسمعة طيبة (تقييم 4.5) وصور حقيقية جذابة للكيك والحلويات، بدون موقع إلكتروني — فرصة واضحة لحضور رقمي يجمع الأصناف والصور والموقع والتواصل في رابط واحد.

## فكرة التصميم
هوية "مخبز دافئ وحلو": كريمي دافئ (#F6ECDF) + بني شوكولاتة (#5A3826) + لمسات توتي/وردي (#C24E6B) مع ذهبي. خط العناوين "El Messiri" والنص "Tajawal". أجواء احتفالية شهيّة وراقية تليق بالمناسبات.

## التقنيات
- HTML / CSS / JavaScript خام (vanilla) — بدون أي إطار عمل أو خطوة بناء.
- خطوط Google عبر `<link>` مع preconnect.
- تأثيرات حركية بـ IntersectionObserver و CSS فقط (بدون مكتبات).

## التشغيل محليًا
```bash
cd cake-alarab
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

## حالة الـ backend
غير مربوط بنظام خلفي. نموذج الطلب **تجريبي**: يجهّز رسالة واتساب جاهزة عبر `wa.me` ويحفظ نسخة في `localStorage` للمتصفح فقط، مع إشعار (toast) للتأكيد.

## مصدر الصور
صور حقيقية من ملف النشاط على خرائط قوقل (كيك العرب). تم اختيار أفضل صور الكيك والحلويات وواجهة المحل واستبعاد ما لا يخص النشاط.
